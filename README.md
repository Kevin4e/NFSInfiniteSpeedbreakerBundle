# Installation:
1. Download the latest version of the [Ultimate ASI Loader](https://github.com/ThirteenAG/Ultimate-ASI-Loader).
2. Copy or move `dinput8.dll` into the root folder of the game.
3. Download this mod from the [releases page](https://github.com/Kevin4e/NFSInfiniteSpeedbreakerBundle/releases).
4. Copy the script for the game into the game's `scripts` folder.
